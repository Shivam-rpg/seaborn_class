# 📊 Seaborn Learning Series (Class 1–5)

Welcome to the **Seaborn Learning Series** repository!
This project is designed for beginners who want to learn **data visualization using Python's Seaborn library** step by step.

---

## 📚 About the Project

This repository contains **5 Jupyter Notebooks**, each focusing on different concepts of Seaborn.
It starts from the basics and gradually moves towards more advanced visualization techniques.

Seaborn is a powerful Python library built on top of Matplotlib that helps in creating **beautiful and informative statistical graphics**.

---

## 🗂️ Course Contents

### 📘 Class 1 – Introduction to Seaborn

* What is Seaborn?
* Installation and setup
* Basic plots
* Working with datasets

---

### 📘 Class 2 – Distribution Plots

* Histograms
* KDE (Kernel Density Estimation)
* Distplots
* Understanding data distribution

---

### 📘 Class 3 – Categorical Plots

* Bar plots
* Count plots
* Box plots
* Violin plots
* Comparing categories

---

### 📘 Class 4 – Relational Plots

* Scatter plots
* Line plots
* Hue, size, and style parameters
* Visualizing relationships between variables

---

### 📘 Class 5 – Advanced Visualizations

* Heatmaps
* Pair plots
* Correlation matrix
* Customizing plots
